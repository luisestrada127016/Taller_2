# 🗂️ Sistema de Inventario en Python

## 📄 Taller_2 – Sistema de Inventario
Sistema de Inventario en Python

## 📌 Descripción
Este proyecto implementa un sistema de inventario básico que permite:
- Agregar productos con nombre, precio y cantidad.
- Listar todos los productos registrados.
- Actualizar la cantidad de un producto existente.
- Eliminar productos del inventario.
- Calcular el valor total del inventario.
- Calcular el valor total de un producto específico.

Los datos se guardan en un archivo `inventario.json` en formato **diccionario**, asegurando persistencia entre ejecuciones.

---

## 📁 Estructura
- `inventario.py` → Código principal del sistema.
- `inventario.json` → Archivo donde se guardan los productos en formato diccionario.
- `README.md` → Documentación del proyecto.

## ▶️ Uso
1. Ejecuta el programa.
2. Selecciona una opción del menú:
   - 1 → Agregar producto
   - 2 → Listar productos
   - 3 → Actualizar cantidad
   - 4 → Eliminar producto
   - 5 → Calcular valor total del inventario
   - 6 → Calcular valor de un producto específico
   - 7 → Salir

## 🚀 Instalacion
1. Clona este repositorio o descárgalo en tu computadora.
2. Asegúrate de tener instalado **Python 3.x**.
3. Ejecuta el programa desde la terminal:
   ```bash
   python inventario.py
